# Vejeeo TechVids

A YouTube Tech Video Data Pipeline for extracting YouTuber information, top videos, and analytics with search and link-passing capabilities.

## Overview

This project provides tools and specifications to:

- **Extract YouTuber metadata** - Channel info, subscribers, descriptions, categories
- **Fetch top/popular videos** - Most viewed, recent, and engaging content
- **Gather analytics** - Growth tracking, engagement metrics, upload frequency
- **Enable search functionality** - Filter by category, language, subscriber count, tags
- **Pass links to external apps** - Export in JSON, CSV, OPML formats with webhook support
